# Usage

Open Copilot Chat in VS Code and select an agent from the agent dropdown.

Agent roles:
- `witch-king`: planning and orchestration (inspect-only).
- `morgul`: backend and API changes.
- `khamul`: frontend and UI work.
- `fellbeast`: review, testing, and release checks.

Recommended workflow:
1. Run `witch-king` to produce a minimal plan.
2. Run `morgul` or `khamul` as appropriate to implement changes.
3. Run `fellbeast` for final validation and release readiness.
