Purpose: Fix build errors following a disciplined approach.

Instructions (for Copilot agent use):
1. Inspect the exact build error output first.
2. Identify the root cause; do not guess.
3. Apply the smallest correct fix; avoid broad or random changes.
4. Re-run or recommend the appropriate validation command (e.g., `dotnet build`, `npm run build`).
5. After fix, request `fellbeast` to run a review and include its validation outputs in the report.
